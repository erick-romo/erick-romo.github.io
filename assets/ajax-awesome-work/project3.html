StateLess
by Code for Progress
StateLess is a web application that allows state and regional leaders to utilize SMS messaging to reach volunteers during political canvassing. This project was completed in collaboration with fellow Code for Progress Fellows. I contributed with the functions page of the project. The back-end for this project was in Python and SQL and Bootstrap for the front-end.
